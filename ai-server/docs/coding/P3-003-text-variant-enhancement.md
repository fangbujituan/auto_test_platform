# P3-003: 文本变体策略增强

> 完成日期：2026-03-24

## 原来实现逻辑

`suitable_text_alternatives` 只处理两种文本变体：

1. 去掉开头的数字部分（如 "3. 设置" → "设置"）
2. 去掉结尾的数字部分（如 "价格 100" → "价格"）

未处理常见 UI 文本模式：
- 括号内容：`"保存 (Ctrl+S)"` → 应匹配 `"保存"`
- 省略号：`"更多..."` → 应匹配 `"更多"`
- 数量词：`"删除 3 项"` → 应匹配 `"删除"`

## 当前实现逻辑

新增三种文本变体策略：

### 策略 3: 括号内容剥离
```python
bracket_stripped = re.sub(r'\s*[\(（][^)）]*[\)）]', '', text).strip()
```
- 支持中英文括号：`()` 和 `（）`
- 示例：`"保存 (Ctrl+S)"` → `"保存"`、`"提交（草稿）"` → `"提交"`

### 策略 4: 省略号剥离
```python
ellipsis_stripped = re.sub(r'[.。…]+$', '', text).strip()
```
- 支持英文点号、中文句号、省略号字符
- 示例：`"更多..."` → `"更多"`、`"查看全部…"` → `"查看全部"`

### 策略 5: 数量词剥离
```python
quantity_stripped = re.sub(r'\s*\d+\s*(项|个|条|件|次|份|张|页|items?|records?|rows?)\s*$', '', text).strip()
```
- 支持中文量词：项、个、条、件、次、份、张、页
- 支持英文量词：items、records、rows
- 示例：`"删除 3 项"` → `"删除"`、`"Selected 5 items"` → `"Selected"`

## 主要修复点

| 修复项 | 说明 |
|--------|------|
| 括号内容剥离 | 中英文括号，score_bonus=1 |
| 省略号剥离 | 英文点号+中文句号+省略号字符，score_bonus=1 |
| 数量词剥离 | 中文 8 种量词 + 英文 3 种量词，score_bonus=2 |
| 最小长度保护 | 剥离后文本 < 2 字符则不生成变体 |
