# 测试用例管理模块快速访问指南

## ✅ 配置检查

运行以下命令检查配置是否正确：

```bash
python check_setup.py
```

如果看到"所有检查通过"，说明配置正确，可以继续。

## 🔧 故障排查

### 问题1：找不到"项目列表"按钮

**解决方案**：
- 检查是否已登录
- 刷新页面
- 查看顶部导航栏右侧

### 问题2：项目列表为空

**解决方案**：
- 点击"新建项目"创建项目
- 或使用演示脚本创建示例数据：
  ```bash
  python test_case_module_demo.py
  ```

### 问题3：点击"测试用例"没反应

**解决方案**：
1. 打开浏览器开发者工具（F12）
2. 查看Console标签是否有错误
3. 检查Network标签，看API请求是否成功
4. 确认后端服务正在运行

### 问题4：页面显示空白

**解决方案**：
1. 确认前端服务正在运行：
   ```bash
   cd client
   npm run dev
   ```
2. 确认后端服务正在运行：
   ```bash
   python run.py
   ```
3. 检查浏览器控制台错误信息
