# 测试用例管理三栏布局实现指南

## 概述
已成功将测试用例管理页面改造为三栏布局，与Bug管理模块保持一致的UI风格。

## 新文件
- `client/src/views/TestCaseManagement2.vue` - 新的三栏布局测试用例管理页面

## 三栏布局结构

### 左侧栏 (200px)
- 项目信息展示
- 功能模块菜单：
  - 接口管理
  - Bug管理
  - 自动化测试（禁用）
  - 测试用例（当前）
  - 环境配置（禁用）

### 中间栏 (350px)
- 搜索框
- 操作按钮（新建模块/新建用例）
- 树形结构展示：
  - 模块节点（带图标和统计信息）
  - 用例节点（带优先级和状态标签）
- 节点操作菜单

### 右侧栏 (自适应)
- 用例详情展示
- 包含以下信息：
  - 基本信息（编号、状态、优先级、类型）
  - 用例描述
  - 前置条件
  - 测试步骤
  - 预期结果
  - 绑定API
- 操作按钮（编辑、删除）

## 主要功能

### 模块管理
- 创建模块（支持父子关系）
- 编辑模块
- 删除模块
- 模块树形展示

### 用例管理
- 创建用例
- 编辑用例
- 删除用例
- 用例详情查看
- 优先级：P0、P1、P2、P3
- 类型：功能、性能、安全
- 状态：草稿、已评审、已废弃

### 交互功能
- 点击节点查看详情
- 右键菜单操作
- 搜索功能（待实现）
- 模块切换导航

## 路由配置
已添加新路由：
```javascript
{
  path: '/projects/:projectId/test-cases-new',
  name: 'TestCaseManagement2',
  component: TestCaseManagement2,
  meta: { requiresAuth: true }
}
```

## 访问方式
1. 通过URL直接访问：`/projects/{projectId}/test-cases-new`
2. 通过左侧菜单切换到"测试用例"模块

## 样式特点
- 渐变色头部（紫色渐变）
- 悬停效果
- 响应式布局
- 自定义滚动条
- 统一的配色方案

- 一致的操作交互
- 统一的视觉风格
- 相似的功能模块切换
